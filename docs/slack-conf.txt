⏺ Slack App 설정 가이드

  1. App 생성

  - https://api.slack.com/apps → Create New App → From scratch
  - App 이름(예: Claude Bot)과 Workspace 선택

  2. Socket Mode 활성화

  - 좌측 Socket Mode → Enable Socket Mode ON
  - Token Name 입력 → scope connections:write → Generate
  - xapp- 으로 시작하는 App-Level Token 복사 → .env의 SLACK_APP_TOKEN에 입력

  3. Bot Token Scopes

  좌측 OAuth & Permissions → Bot Token Scopes에 5개 추가:

  ┌───────────────────┬──────────────────┐
  │       Scope       │       용도       │
  ├───────────────────┼──────────────────┤
  │ app_mentions:read │ @멘션 읽기       │
  ├───────────────────┼──────────────────┤
  │ chat:write        │ 메시지 전송      │
  ├───────────────────┼──────────────────┤
  │ im:history        │ DM 히스토리 읽기 │
  ├───────────────────┼──────────────────┤
  │ im:read           │ DM 읽기          │
  ├───────────────────┼──────────────────┤
  │ im:write          │ DM 전송          │
  └───────────────────┴──────────────────┘

  4. Event Subscriptions

  좌측 Event Subscriptions → Enable Events ON

  Subscribe to bot events에 2개 추가:
  - app_mention
  - message.im

  5. App 설치

  좌측 Install App → Install to Workspace → 권한 승인
  → xoxb- 으로 시작하는 Bot Token 복사 → .env의 SLACK_BOT_TOKEN에 입력

  6. DM 허용 (선택)

  좌측 App Home → Messages Tab 체크
  → "Allow users to send Slash commands and messages from the messages tab" 체크

  ---
  채널 ID 찾는 법

  Slack에서 채널 이름 클릭 → 팝업 하단에 채널 ID (C로 시작) 표시

  토큰 입력 후 실행
